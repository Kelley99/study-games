# 女儿闯关游戏

七年级下册历史 / 生物 / 地理三科闯关复习游戏。

## 游戏

| 学科 | 文件 | 关卡 | 题量 |
|------|------|------|------|
| 历史 | [history-adventure/index.html](history-adventure/index.html) | 8关 | 192题 |
| 生物 | [biology-adventure/index.html](biology-adventure/index.html) | 4关 | 96题 |
| 地理 | [geography-adventure/index.html](geography-adventure/index.html) | 6关 | 144题 |

## 访问方式

启用 GitHub Pages 后，访问：
- `https://Kelley99.github.io/study-games/history-adventure/`
- `https://Kelley99.github.io/study-games/biology-adventure/`
- `https://Kelley99.github.io/study-games/geography-adventure/`

## 本地运行

```bash
# 历史（端口8898）
cd history-adventure && python3 -m http.server 8898

# 生物（端口8897）
cd biology-adventure && python3 -m http.server 8897

# 地理（端口8896）
cd geography-adventure && python3 -m http.server 8896
```

用浏览器打开 `http://localhost:8898` 等即可。
